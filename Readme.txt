Đây là project quản lý sinh viên dùng mySQL


Cấu trúc mySQL: sv
info: chứa MSSV, tên, ngày sinh, mail, khoa, sđt
login: chứa type, tài khoản, mật khẩu, MSSV
Vào thư mục myphp để bắt đầu -> index.php
Cấu trúc gồm file index.php -> login.php.
Chưa có tài khoản có thể đăng ký dưới dạng sinh viên.
Khi login dưới dạng sinh viên -> xem và sửa thông tin cá nhân.
Khi login dưới dạng admin sẽ có thể truy vấn, xem, chỉnh sửa và xoá thông tin học sinh.
Đăng xuất sẽ trở về login.php ban đầu.
